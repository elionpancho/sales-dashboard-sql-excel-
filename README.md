# sales-dashboard-sql-excel-
End-to-end sales data analysis using SQL and Excel, featuring data cleaning, pivot tables, and interactive dashboard visualization.


![grabeight dashboard analysis](https://github.com/user-attachments/assets/4762d948-723b-4419-bbff-5307de9e3d42)


**Sales Dashboard (SQL + Excel)**

A data analytics project that uses SQL for data extraction and Excel for data cleaning, transformation, and visualization through pivot tables and dashboards.

**Tech Stack**

*SQL – data querying and extraction

*Excel – data cleaning, pivot tables, and dashboard creation

**Features**

*Total revenue, monthly, and daily sales breakdown

*Sales trends visualization (daily & monthly charts)

*Top-selling food items

*Top beverages, canned goods, and ice cream

*Customer peak days and hours

*Interactive dashboard using Excel

**Dashboard Preview**

**Data Process**

*Extracted raw sales data using SQL queries

*Cleaned and transformed data in Excel

		1.Removed duplicates
		
		2.Handled missing values
		
		3.Standardized formats

*Created Pivot Tables for analysis

*Built an interactive dashboard using charts and slicers

**Key Insights**

*Peak customer time occurs at 12 PM

*Average customer interval is 20 minutes

*Certain product categories (e.g. beverages and ice cream) dominate sales

*Monthly sales trends show fluctuations that can guide inventory decisions

**How to Use**

*Download the Excel file from this repository

*Open in Microsoft Excel

*Explore the dashboard

**Purpose**

*This project was created to:

*Practice SQL data extraction

*Improve Excel data cleaning and pivot skills

*Build a real-world business dashboard
